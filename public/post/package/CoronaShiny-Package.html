In this blog I will be talking about packages, what do they entail and what is a good package like and then we will be reviewing a package made by a colleague of mine. The name of the package is CoronaShiny. The author of the Package is Vinny Vu. You can check it out on git. The package is wrapped around the corona virus pandemic and has an inbuilt shiny-app inside the package. We will discuss what the package entails a bit later on.

What did I think this package would do? Did it do it?

The idea I got for what the package contains and what I can use from it, is that the package contains an app helping users interact and get visulaizations on the cumulative cases of corona-virus in each continant.

Are the improvements that could be made to the code? Are there improvements that could be made to the shiny app inside the package? Is the documentation sufficient to get started?
