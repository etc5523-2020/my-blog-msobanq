Romania & CoVID-19

Last updated on Sep 2, 2020 6 min read 
c19 <- read.csv("owid-covid-data.csv")

DATA Description

The data sets used are of the COVID-19 virus, containing data on all countries. The first data set contains the total cases, new cases, cases per million of the countries population, total deaths, new deaths, new tests, total tests and stingency index. The data set has extensive variables, however, not all are used in this blog. The data set was obtained form here. The first set although extensive, did not conatin the varaible of cases recovered. A second data set was used to obtain the cases recovered from here.

Why Romania?

Romania <- c19 %>% filter(location == "Romania") %>% 
  select(-iso_code, 
         -continent, 
         -location) %>% 
  filter(date >     "2020-02-26") %>% mutate(date = as.Date(date))
Romi <- c19 %>% filter(location == "Romania") %>% 
  select( date, new_cases, new_tests) %>% 
  filter(date >     "2020-02-29")  %>% mutate(Month = month(date,label = TRUE,abbr = FALSE)) %>% na.omit() 
rp <- Romi %>% ggplot( aes(x= new_tests, 
                           y = new_cases, 
                           color = Month)) + 
  geom_point() # + facet_wrap(~ month(date)) + geom_smooth()
